# Advanced Calculator

A stylish, fully functional calculator built with pure HTML, CSS, and JavaScript — no frameworks, no dependencies.

## Live Demo

Open `task2.html` directly in any browser. No server required.

## Features

- Basic arithmetic — addition, subtraction, multiplication, division
- Percentage ( % ) support
- Backspace ( ⌫ ) to delete last character
- Clear ( C ) to reset display
- Keyboard support — type numbers and operators directly from your keyboard
- Auto Color theme toggle — enables a continuous hue-shift animation on the calculator
- Operator buttons styled in a distinct orange gradient to stand out from number buttons
- Glassmorphism card design with backdrop blur

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `0–9` | Input digits |
| `+ - * / . %` | Input operators |
| `Enter` | Calculate result |
| `Backspace` | Delete last character |
| `Escape` | Clear display |

## Background Animations

- Aurora — two spinning conic gradient layers that slowly rotate and breathe
- Floating glow orbs — three radial gradient blobs that float upward and fade
- Twinkling starfield — 160 stars rendered on a canvas that twinkle independently

## Tech Stack

| Technology | Usage |
|------------|-------|
| HTML5 | Structure and canvas element |
| CSS3 | Glassmorphism, animations, grid layout |
| JavaScript (Vanilla) | Calculator logic, keyboard events, starfield canvas |
